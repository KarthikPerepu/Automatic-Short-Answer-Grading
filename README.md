# Automatic Short Answer Grading

An AI-powered application that automatically evaluates and grades students' short-answer responses using Natural Language Processing (NLP) and Machine Learning techniques. The system analyzes the semantic similarity between a student's answer and the reference answer to generate accurate and consistent scores, reducing the need for manual evaluation.

## Features

- Automated grading of short-answer responses
- NLP-based text preprocessing and semantic analysis
- Machine Learning model for score prediction
- Consistent and efficient evaluation
- Easy to extend with advanced language models

## Tech Stack

- Python
- NumPy
- Pandas
- Scikit-learn
- NLTK
- Matplotlib

## Getting Started

```bash
git clone https://github.com/KarthikPerepu/Automatic-Short-Answer-Grading.git
cd Automatic-Short-Answer-Grading
pip install -r requirements.txt
python main.py
```
